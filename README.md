🍔 OrderPick

OrderPick is a Java Swing–based desktop food ordering application with MySQL integration. The system allows customers to browse menu items, manage a cart, and complete secure payments, while administrators can monitor live orders and manage system data.

🚀 Key Features
1- Role-Based Access: Separate interfaces for customers and administrators.
2- Authentication System: User registration and login connected to a MySQL database.
3- Dynamic Menu: Categorized items such as burgers, sides, and drinks.
4- Checkout System: Credit card validation with expiry and format checks.
5- Digital Receipts: Generates unique order IDs and pickup codes with an option to export receipts.
6- Admin Dashboard: Monitor orders, menu items, and administrator data.

🛠️ Tech Stack
- Language: Java
- GUI: Swing / AWT
- Database: MySQL
- Architecture: Object-Oriented Programming (Encapsulation, Inheritance, Custom Exceptions)
